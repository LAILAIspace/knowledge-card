人口老化的長壽社會，是災禍還是禮物

打開媒體，整個社會每天在談，長壽社會來了！健保快破產了！退休基金撐不久了！年輕人的扶養比受不了了！

人們長壽活得久，大家好像都很悲觀，很少從禮物的角度來看待這件事。

未來人生已經沒有老這件事情了，人生就是一個人不斷向前走的一條路

北歐很多國家推動的高齡發展目標之一，就是把臥床縮短到兩週，你可能看到一個90幾歲的老人因為疾病或衰竭，躺著準備離開的時候，你發現他有六塊肌，他可能也背著書包，搞不好他在讀大學，他也打電腦，可能還在工作，搞不好他還在談戀愛喔！如果像這樣，那麼高齡社會就是上帝給人類最棒的禮物了！

Github 將程式碼打包送往北極保存

知名開源平台 Github 宣布將存放於平台的開源程式碼送往北極保存

為何需要大費周章將程式碼打包起來，運送到距離歐洲大陸有 1,000 公里遠的北極？

為了避免產生資料遺失的風險，他們將 21 TB的原始碼轉成 QR Code 底片，以膠捲形式存放於鋼盒中，運往北極的世界資料檔案館保存。北極天氣寒冷檔案不易變質，有利於存放資料。

程式碼諾亞方舟計畫，將世界各地工程師的心血集合起來妥善保存，作為人類文明的紀錄。
